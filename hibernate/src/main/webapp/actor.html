Exercise Requirements

  • Create Account or User Registration Form
  • First Name, Last Name, Email Address, Password, Confirm Password
  • Submit button that when submits executes a validate function.
  • Validation function can be done as an onsubmit in the form tag or by adding an event listener
  • Recommendation is using Bootstrap and jQuery but not required
  • When the user corrects their input mistakes, remove any of the error formatting and border color

Validation Requirements

  • All fields are required. Do not use the required attribute on the HTML tag as we want to do with JavaScript
  • First name field should not contain a space. Bonus : Achieve this with a regex in javascript
  • Optional : find a prebuilt regex on the internet and use it to validate the email address is proper.
  • Password and Confirm password are both required and must be the same. This is 2 different error messages
  • Password must contain a min of 8 characters and a max of 20 characters.
  • Password must contain a letter and a number
  • If all validates properly show an alert that says something nice.

Error Messages

  • Each field that has an error, use CSS to make the border of the input box red
  • Under each input box, create / insert HTML / CSS to show an error message of what is wrong with the input
  • Error messages should be red and maybe a slightly smaller font
  • Try to make the message properly left aligned with the input box. (optional)
  • Extreme Stretch : execute the password validation check on each key press and alter the error messages as the password is being typed.
